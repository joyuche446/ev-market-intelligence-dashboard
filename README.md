## Executive Summary:
This project presents an in-depth analysis of electric vehicle (EV) adoption and performance trends using a dataset of over 150,000+ vehicle records. Built using Tableau, the interactive dashboard provides a comprehensive view of the EV market, including total vehicle population, average electric range, technology distribution (BEV vs PHEV), manufacturer dominance, model year trends, geographic distribution, and Clean Alternative Fuel Vehicle (CAFV) eligibility.
The analysis reveals strong and consistent growth in EV adoption, particularly after 2020, with Battery Electric Vehicles (BEVs) accounting for over 80% of the market, indicating a clear shift toward full electrification.
Overall, this dashboard enables stakeholders to understand EV market dynamics and identify key adoption patterns supporting the transition to sustainable mobility.
